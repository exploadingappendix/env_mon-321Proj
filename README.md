# env_mon-321Proj
CSE 321 - Enviornment Monitor 

# SETUP

To Set up the Enviornment Monitor, connect it to a local device or power source and it should begin to display information about the local enviornment. 

The Enviornment Monitor Displays: 
Humidity information
Air Quality, 
temperature.

----

# Interaction

Once connected, the Enviornment monitor displays humidity and temperature information. Humidity and Air Quality information is switched using a timer, once the timer goes off the data not being displayed replaces the data being displayed. For example if Humidity is being displayed, after some ms, the display changes to displaying air quality information.
